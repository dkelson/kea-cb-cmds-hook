# Notice

This project implements the publicly documented Kea Configuration Backend
Commands API for `libdhcp_cb_cmds.so`.

It is not affiliated with, sponsored by, or endorsed by Internet Systems
Consortium. It is not a copy of ISC's premium hook implementation.

Kea is a trademark of Internet Systems Consortium.

The build includes a Bison-generated `location.hh` compatibility header used
only when an installed Kea development package omits `eval/location.hh` while
still installing `eval/parser.h`. Bison grants a special exception allowing
larger works containing that parser skeleton output to be distributed under
terms of the larger work.
